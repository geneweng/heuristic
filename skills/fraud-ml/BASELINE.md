# fraud-ml baseline

Generated by `python skills/fraud-ml/train.py`. Frozen for the rest of the
POC per the HL thesis — rules adapt while ML stays still.

## Algorithm

`sklearn.ensemble.HistGradientBoostingClassifier` · 200 iters · depth 6

(Choice over XGBoost: HistGBM has no native deps and matches XGBoost
performance on the synthetic data this POC uses.)

## How to retrain

```bash
make data && make ml-train
```

`make data` regenerates the splits from `RANDOM_SEED=42`; `make ml-train`
fits the model and overwrites `skills/fraud-ml/model.joblib`. Output is
deterministic given the seeds.
